# Доступность облигации (Bond Availability Check)

## Оригинальное описание

> Проверка доступности определенной облигации по прямому запросу. У меня есть список облигаций, доступных на бирже, но мой провайдер облигаций может ещё не разместить у себя одну из облигаций, которая мне интересна. Поэтому мне нужен агент, который каждый день делал бы запрос на определенный url и узнавал, залистил ли провайдер нужную облигацию. Установили, когда надо. Удалили по завершению.

## Анализ

### Реализация (L1)

`manage_schedule` (daily cron) + `http_request` (GET на URL провайдера) + промпт для парсинга ответа. Если облигация найдена — `send_message`. Если нет — молчит или "пока нет".

### Lifecycle — временный агент

Интересный паттерн: install → работает каждый день → цель достигнута → uninstall. Это агент с конечным временем жизни.

Нет механизма "self-uninstall" — агент не может сам себя удалить (manage_agents — privileged). Варианты:
- Master видит результат и предлагает: "Облигация найдена, удалить агента?"
- Промпт агента: "Когда цель достигнута, скажи пользователю что задача выполнена и спроси, удалять ли агента"
- Через `switch_to_agent('master')` с сообщением "Цель достигнута, нужно удалить меня"

## Что есть в системе

| Компонент | Статус |
|-----------|--------|
| `manage_schedule` tool | Готов |
| `http_request` tool | Готов |
| Cronos → webhook → agent pipeline | Готов |
| `manage_schedule` в CAPABILITY_TOOLS | Готов |
| Self-uninstall для агентов | Нет — workaround через Master |

## Что нужно для реализации

1. Настроить endpoint в manifest: URL провайдера
3. Написать промпт (проверь URL, если найдено — сообщи, предложи удалить агента)
4. `manage_agents.install(manifest)` + `manage_schedule.create(daily cron)`

## Предварительная оценка

| Аспект | Значение |
|--------|----------|
| Уровень | L1 |
| Tools | `manage_schedule`, `http_request` |
| Proactive | Да — ежедневный scheduler |
| Lifecycle | Временный (install → goal → uninstall) |
| Готовность системы | ~95% (продумать lifecycle) |
| Сложность реализации | Низкая |
